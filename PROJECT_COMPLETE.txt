🎮 GAME FEATURES (COMPLETE)
═══════════════════════════════════════════════════════════════════════════

Core Mechanics:
  ✅ 2-6 players with custom names
  ✅ 3×3 tic-tac-toe board per player
  ✅ 4 hearts per player (lives system)
  ✅ Fair RNG dice (crypto.getRandomValues)
     - Green: 2 faces (1/3 odds)
     - Red: 2 faces (1/3 odds)
     - Yellow: 2 faces (1/3 odds)
  ✅ Up to 3 rolls per turn
  ✅ Triple detection with free roll bonus
  ✅ Wild card menu (4 actions)
  ✅ Win detection (8 tic-tac-toe patterns)
  ✅ Heart loss on failed rolls
  ✅ Game over detection (last player standing)
  ✅ Clockwise turn order

Game Flow:
  1. Setup: Select players, enter names
  2. Play: Roll dice → Keep/reroll → Check for triple
  3. Place: Green/Red/Wild card actions
  4. Check: 3-in-a-row? All others lose 1 ❤️
  5. Win: Last player with ❤️ = CHAMPION 🏆

🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Install & Start:
   $ npm install --legacy-peer-deps
   $ npm run web
   → Open http://localhost:8081

2. Test Locally:
   $ npm run type-check   # TypeScript ✅
   $ npm run build:web    # Production build

3. Deploy:
   Option A (Vercel - Easiest):
     $ git push origin main
     
   Option B (Docker):
     $ docker build -t aon .
     $ docker-compose up
     → http://localhost:8080
     
   Option C (Self-hosted):
     $ bash deploy.sh
     $ scp -r dist/* user@server:/var/www/aon/
